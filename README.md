## Installation

> **Manual installation:**  

1. **Download** this project (clone or `.zip` extract).
2. Open your browser and go to: `chrome://extensions/`
3. Enable **Developer Mode** (top right toggle).
4. Click **Load unpacked** and select the project folder.
